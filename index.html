Aprendiendo HTML

Conoce las etiquetas más comunes

<p> - Párrafo

La etiqueta <p> se usa para escribir párrafos de texto.

<a> - Enlace

La etiqueta <a> sirve para crear enlaces a otras páginas.

<img> - Imagen

Muestra una imagen en tu página.

<ul> y <li> - Lista

Sirve para hacer listas sin orden.
